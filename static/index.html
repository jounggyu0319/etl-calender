동기화

자주 쓰는 순서대로 배치했어요.

캘린더 구독 URL 기준으로 빠르게 반영해요.

과제·퀴즈까지 가져올 때는 먼저 eTL 로그인 후 전체 동기화를 눌러 주세요.

연결 설정

Google 캘린더

1단계

캘린더 구독 URL

간편 동기화용

myetl 캘린더에서 «URL 주소 가져오기»로 복사한 주소를 붙여 넣어요. 비우고 저장하면 삭제돼요.

eTL 계정 (선택)

전체 동기화용

과제·퀴즈까지 자동으로 가져올 때만 필요해요. MFA를 줄이려면 Chrome을 --remote-debugging-port=9222로 실행한 뒤 로그인하고, 서버에서 ETL_HEADLESS=false일 때 127.0.0.1:9222에 붙는 방식을 쓸 수 있어요.

Chrome 확장 프로그램

권장

이미 myetl에 로그인된 탭에서 클릭 한 번으로 동기화할 수 있어요.

확장 설치 안내 보기 ▼
  1. extension/ 폴더를 Chrome에 압축 해제 설치
  2. 팝업에서 이 계정으로 로그인
  3. myetl 탭에서 동기화 클릭

참고: 데스크톱 앱 방식은 저장소의 desktop/을 확인해 주세요.
