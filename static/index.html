📚 eTL 과제·일정 자동 캘린더 등록

eTL에 올라온 과제, 퀴즈, 시험 공지를 Google 캘린더에 자동으로 추가해 드려요. 로그인 한 번으로 매일 자동 동기화됩니다.

가입 · 로그인

설정 순서

1단계 — Google 캘린더 연결

2단계 — eTL 연결 (구독 URL · 계정)

캘린더 구독만 쓰면 간편 동기화만으로도 되고, 과제·퀴즈까지 하려면 아래에 eTL 아이디를 저장한 뒤 로그인 버튼을 쓰세요.

myetl 캘린더 → «URL 주소 가져오기»로 복사한 주소를 붙여 넣고 저장합니다. 비우고 저장하면 삭제됩니다.

eTL 아이디·비밀번호 (선택)

과제·퀴즈까지 자동으로 가져올 때만 필요합니다. 통합인증(MFA)을 줄이려면 Chrome을 --remote-debugging-port=9222 로 먼저 실행해 로그인한 뒤, ETL_HEADLESS=false·Chrome일 때 서버가 127.0.0.1:9222 에 붙습니다. 그 외에는 동기화 시 새 창이 뜰 수 있고, 창이 보이는 설정에서는 끝난 뒤에도 자동으로 닫지 않을 수 있습니다.

3단계 — 동기화 시작

eTL 캘린더에 등록된 일정만 가져옵니다. 빠르고 간단해요.

과제·퀴즈·공지까지 전부 가져오려면 먼저 로그인하세요.

과제, 퀴즈, 공지사항까지 전부 캘린더에 추가합니다.

구독 URL만 있으면 위 첫 버튼만으로도 됩니다. eTL 계정을 저장했다면 「🔐 eTL 로그인」으로 브라우저를 연 뒤 로그인·MFA를 마치고 「🔄 전체 동기화 시작」을 눌러 주세요.

확장·데스크톱

이미 myetl에 로그인한 브라우저에서만 쓰는 방식입니다. 저장소의 extension·desktop 폴더를 참고하세요.
