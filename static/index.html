연결 설정

Google 캘린더

1단계
⚠️ 연결 시 경고창이 뜨면?

Google 계정 연결 버튼을 누르면 「Google에서 확인하지 않은 앱」 경고 화면이 나타날 수 있어요.

아래 순서대로 진행하면 정상 연결됩니다.

  1. 경고 화면 하단 「고급」 클릭
  2. 「(앱 이름)(으)로 이동」 클릭
  3. 캘린더 접근 권한 허용

이 메시지는 앱이 Google 공식 검수를 받기 전 소규모 운영 중이라 표시돼요. 기능·보안에는 문제 없습니다.

기본 일정 링크

마감일 있는 일정
? 기본 일정 링크 발급 방법

eTL에 있는 마감일 일정을 자동으로 가져오는 개인 링크예요.

  1. myetl 캘린더 접속
  2. 우측 하단 캘린더 피드 클릭
  3. 표시된 URL 복사 후 아래에 붙여넣기

비우고 저장하면 삭제돼요.

공지·과제 동기화 키

공지·퀴즈·과제 전부
? 공지·과제 동기화 키 발급 방법

서버가 내 계정으로 공지·과제·퀴즈를 가져올 수 있게 해주는 개인 키예요. 비밀번호 자체는 저장하지 않아요.

  1. myetl 설정 접속 → + 새 액세스 토큰 클릭
  2. 목적에 아무 이름 입력 (예: etl-calendar)
  3. 만료일은 비워두기 → 토큰 생성
  4. 표시된 토큰 복사 후 아래에 붙여넣기 (창 닫으면 다시 볼 수 없어요)

비우고 저장하면 삭제됩니다.
