Canvas API 토큰

전체 동기화(서버)

myetl 설정에서 토큰을 발급한 뒤 붙여 넣고 저장하세요. 저장하면 Render 등 클라우드에서도「전체 동기화」가 서버 Canvas API로 동작합니다. 비우고 저장하면 삭제됩니다.

Chrome 확장 프로그램

권장

이미 myetl에 로그인된 탭에서 클릭 한 번으로 동기화할 수 있어요.

확장 설치 안내 보기 ▼
  1. extension/ 폴더를 Chrome에 압축 해제 설치
  2. 팝업에서 이 계정으로 로그인
  3. myetl 탭에서 동기화 클릭

참고: 데스크톱 앱 방식은 저장소의 desktop/을 확인해 주세요.
