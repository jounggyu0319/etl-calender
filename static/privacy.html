eTL Calendar Sync

개인정보처리방침

최종 업데이트: 2026년 4월 21일

eTL Calendar Sync(이하 "서비스")는 서울대학교 eTL(Moodle 기반 LMS)의 과제·일정을 Google 캘린더와 동기화하는 개인 프로젝트입니다. 본 방침은 서비스 이용 시 수집·이용되는 개인정보에 대해 안내합니다.

1. 수집하는 정보

  • Google 계정 정보: OAuth 2.0 인증을 통해 이름, 이메일 주소, Google 캘린더 접근 토큰을 수집합니다.
  • eTL/Canvas 정보: 사용자가 직접 입력한 eTL iCal URL 또는 Canvas API 토큰을 저장합니다.
  • 동기화 이력: 동기화 실행 시각 및 결과(성공/실패)를 로그로 저장합니다.

2. 정보의 이용 목적

  • Google 캘린더에 eTL 과제·일정 이벤트를 생성·업데이트하기 위해 사용합니다.
  • 자동 동기화 기능 실행을 위해 인증 토큰을 저장합니다.
  • 서비스 품질 개선 및 오류 진단 목적으로 로그를 활용합니다.

3. 정보의 보관 및 보호

  • 인증 토큰 등 민감 정보는 암호화하여 데이터베이스에 저장합니다.
  • 제3자에게 개인정보를 판매하거나 제공하지 않습니다.
  • 서비스 탈퇴 시 관련 데이터를 즉시 삭제합니다.

4. Google API 사용 범위

본 서비스는 Google Calendar API를 통해 캘린더 이벤트를 읽고 쓰는 권한(https://www.googleapis.com/auth/calendar)을 요청합니다. 이 권한은 eTL 일정을 동기화하는 목적에만 사용됩니다.

본 서비스의 Google API 사용은 Google 개인정보처리방침 및 Google API 서비스 사용자 데이터 정책을 준수합니다.

5. 사용자 권리

  • 언제든지 서비스 내 설정에서 Google 계정 연동을 해제할 수 있습니다.
